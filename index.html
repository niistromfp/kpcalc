Прайс и себестоимость ФП-1 по маркам

Единая таблица: цена продажи, полная себестоимость, валовая прибыль и рентабельность для базовой плиты ФП-1 1990×950×150 (0,283575 м³) по каждой марке. Себестоимость = бетон + доска + каркас + производственные расходы. С марки И14 каркас из жаростойкой стали 12Х18Н10Т. Прибыль и рентабельность считаются без НДС. Даты обновления проставляются автоматически при изменении цены.

Доска: на одну ФП-1 идёт 2 шт 950×100×25 мм (0,002375 м³ каждая). Для остальных типоразмеров доска и каркас считаются пропорционально объёму, каркас доборных плит - с надбавкой 15%. Реквизиты организации

Сервис

Данные сохраняются автоматически в браузере (localStorage) - привязаны к конкретному браузеру и компьютеру. Для переноса на другую машину используйте облачную синхронизацию или ручной бекап.

Облачная синхронизация (Google Drive)

Автоматическое сохранение всех данных (история, прайсы, каталог, реквизиты) в служебную папку вашего Google Drive. Работает только при размещении на хостинге (HTTPS), не работает с локального файла. Как настроить?

Однократная настройка (10 минут):
  1. Зайти в console.cloud.google.com, создать проект
  2. В разделе «APIs & Services → Library» включить «Google Drive API»
  3. В «APIs & Services → OAuth consent screen» заполнить базово: User Type = External, status = Testing, добавить себя в Test users
  4. В «Credentials → Create Credentials → OAuth client ID»: type = Web application, в «Authorized JavaScript origins» вписать URL вашего хостинга (например https://username.github.io)
  5. Скопировать Client ID, вставить в поле ниже
  6. Нажать «Войти через Google»

Ручной бекап

Если облачная синхронизация не нужна или не настроена - можно вручную скачать/загрузить JSON-файл со всеми данными.

Загрузка бекапа полностью заменяет текущие данные в этом браузере.
